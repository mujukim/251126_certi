락다운 시티
N*N 격자무늬 도시가 주어진다.
구역은 (행, 열)로 나타낸다. 
모든 구역은 보안등급을 가지고 있으며, 구역에 진입하기 위해서 해당구역의 보안등급보다 같거나 높은 보안등급을 가진 통행증 필요.
보안등급은 대문자로 이루어진 단어로 죄대길이는 M을 넘지 않으며, 
사전순서가 더 빠를수록 높은 등급이다.
통행증은 사용횟수와 보안등급을 지정해서 발급 받을 수 있으며
상하좌우 인접해있는 지역에 진입할 때마다 사용횟수가 1회 차감된다.
L회 사용 횟수를 가지는 통행증을 발급받을 때 출발구역에서 도착구역까지 이동가능한 경로 중 가장 낮은 보안등급을 요구하는 경로를 찾아야 한다.

void init(int N, int M, char mGrade[][][])
N : 도시크기 5 =< N =< 100
M : 보안등급 최대길이 1=< M =< 3
mGrade : 각구역의 보안등급 mGrade[r][c]

void change(int mRow, int mCol, int mDir, int mLength, char mGrade[])
(mRow, mCol) 구역부터 mDir이 증가하는 방향으로 mLength 개의 구역이 mGrade로 변경한다. 
mDir : 0이라면 행이 증가하는 방향, 1이라면 열이 증가하는 방향

char* calculrate(int L, int sRow, int sCol, int eRow, int eCol)
cnfqkfrndur(sRow,sCol)에서 도착구역(eRow, eCol)까지 L회 사용횟수를 가진 통행증으로 이동가능한 경로 탐색.
이동에 필요한 가장 낮은 등급을 반환해라.
단, 통행증은 다른 구역으로 이동할 때 사용되므로 출발구역은 사용횟수, 최소보안등급에 포함되지 않는다. 
출발과 도착구역은 서로다른위치다.
C++인 경우, 문자열 끝에 '\0'을 넣는다.
